Introduction:¶

In this project, we gather, assess, and clean data and then act on it by wrangling the data.

  • WeRateDogs Twitter archive was downloaded from Ucadity (given to us by downloading it from the classroom)

  • Twitter Image Prediction (Image_predictions.tsv) was downloaded programmatically from the URL using the Requests library

  • Additional data from Twitter was downloaded using Twitter API (Tweepy) and stored as tweets_data.txt

  • Data was then assessed visually and programmatically before being cleaned for exploratory analysis.
